The Virtual Window

I built this app one day when I looked out my window and noticed my view was blocked by trees. I felt like I didnt have anything nice to look at so I built a weather app that takes the clients latitude and longitude coordinates, plugs them into an OpenWeatherMap API Call, and creates a pretty image depicting the weather outside, including weather data and a clock. It also can generate an advice slip via Adviceslip API. Built using Vite.js, React, Hooks, and deployed by Netlify.
